# MD · Preview 技術仕様書

## 1. 概要
このプロジェクトは、単一の HTML ファイルで完結する高精度な Markdown プレビューツールです。
開発者の迅速なドキュメント作成と確認をサポートし、ボタン一つでポータブルな HTML ドキュメントを書き出すことができます。

## 2. 主な機能

### 2.1 エディタ・プレビュー
- **リアルタイムプレビュー**: 入力と同時に Markdown を HTML にレンダリング。
- **Mermaid 対応**: フローチャート、シーケンス図、ガントチャートなどの図表描画をサポート。
- **GitHub スタイルアラート**: `[!NOTE]`, `[!TIP]` 等の拡張記法に対応。
- **シンタックスハイライト**: HLJS による多言語対応のコードハイライト。
- **ユーティリティ**: クリア機能、入力例の挿入、タブ入力（スペース2つ）のサポート。

### 2.2 デザイン・UI
- **システム標準フォント採用**: 外部への依存を無くしオフラインでも正しく表示されるよう、`Meiryo`, `Segoe UI`, `Yu Gothic` 等の標準搭載フォントを使用。
- **ダーク/ライトモード**: ツール全体のテーマ切り替えに対応。
- **レスポンシブ・スプリットビュー**: ドラッグによる画面分割幅の調整。
- **ソース表示モード**: プレビュー画面を切り替えて、HTMLレンダリング前のソースを確認可能。

### 2.3 エクスポート機能
「HTML保存」ボタンにより、単体で動作する高品質なHTMLドキュメントを出力します。
- **自動目次 (TOC)**: `h1` 〜 `h4` を自動抽出し、サイドバーに階層表示。
- **テーマの固定**: ダウンロード時のテーマ（ライト/ダーク）がそのまま適用された状態で出力されます。これにより不要なスタイル定義を排除し、ファイルサイズを削減。
- **軽量化**: 切替ボタンや関連ロジックを削除し、純粋なドキュメント閲覧に特化した構成にしています。
- **システム標準フォント**: エクスポート先でも外部フォントを必要とせず、オフラインで正しく表示されます。
- **レスポンシブレイアウト**: PCでは2カラム（サイドバー＋メイン）、モバイルでは1カラムに自動調整。

## 3. 技術スタック
- **HTML5 / CSS3 / JavaScript (Vanilla)**
- **marked.js**: Markdown パーサー
- **highlight.js**: コードハイライト
- **mermaid.js**: 図表レンダリング

## 4. 使用方法
1. 左側のエディタに Markdown を入力。
2. 右側にリアルタイムでプレビューが表示される。
3. 「入力例表示」から全機能のサンプルを確認可能。
4. 完成したら「HTML保存」で配布用ドキュメントをダウンロード。
