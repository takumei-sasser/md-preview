MD · PREVIEW User Guide

シンプルで強力な、Webベースの Markdown プレビューツールの使い方を解説します。

ツールについて

MD · PREVIEW は、単一の HTML ファイルで動作するエンジニア向けドキュメント作成支援ツールです。インストール不要でブラウザさえあれば、どこでも高精度なプレビューと HTML 書き出しが可能です。

リアルタイムレンダリング 入力した内容が即座に右側のプレビューに反映されます。
ポータブル・デザイン システム標準フォントと軽量なライブラリ構成により、オフラインでも正しく動作します。

はじめかた

  1. Markdown を書く: 左側のエディタエリアに Markdown を入力します。
  2. プレビューを確認する: 右側のプレビューエリアにレンダリング結果が表示されます。
  3. HTML を保存する: 画面右上の「HTML保存」ボタンから、配布用の HTML ファイルをダウンロードできます。
Tip
操作に迷ったら、右上の「入力例表示」ボタンを押してみてください。全ての機能を含んだサンプルコードを挿入できます。

下書きと自動保存

書いた内容を失わないための強力なデータ保護機能を備えています。

自動保存 (Last State) 入力内容は常にブラウザに自動保存されます。タブを閉じたりリロードしても、最後に入力していた状態から再開できます。
名前付き保存 (Drafts) 「保存」ボタンから、現在の内容に名前を付けて「下書き」として保存できます。 タイトルは # 見出し から自動的に推測され、手動での変更も可能です。
下書きの管理: 「下書き」ボタンを押すと、保存済みデータの一覧が表示されます。ここから過去のデータを「読み込み」したり、不要になったものを「削除」できます。

ファイル操作

ローカルにある既存の Markdown ファイルを簡単に編集できます。

  • ファイルを開く: ツールバーの「開く」ボタンからファイルを選択して読み込めます。
  • ドラッグ＆ドロップ: パソコン上の .md ファイルを直接エディタエリアにドロップすることで、素早く内容を読み込めます。

便利なツールバー機能

画面上部のボタン群から、以下の操作が可能です。

  • テーマ切替: ライトモードとダークモードを瞬時に切り替えます。
  • ソース表示: プレビューエリアを Markdown ソースまたはプレビュー表示に切り替えます。
  • クリア: エディタの内容を全て消去します（localStorage の最新状態もリセットされます）。
  • 全ソース表示: このツール自体の HTML コードを表示・コピーできます。

Markdown 記法

標準的な Markdown 記法に加え、以下の機能に対応しています。

  • シンタックスハイライト: プログラミングコードの言語に応じた色分け表示が可能です。
  • テーブル: 表形式のデータを美しく表示します。
  • チェックリスト: - [x] 形式のタスク管理記法に対応しています。

図表の描画 (Mermaid)

Mermaid.js を内蔵しており、コード形式で様々な図表を描画できます。

利用例:

```mermaid
graph TD
A[開始] --> B{判定}
B -->|Yes| C[処理]
B -->|No| D[終了]
```

GitHub スタイルアラート

ドキュメントの中で特に強調したい内容がある場合、GitHub スタイルのアラートが利用可能です。

NOTE
一般的な情報や補足説明。
TIP
役立つヒントや推奨事項。
WARNING
注意が必要な事項。
DANGER / CAUTION
重大なリスクや警告（CAUTION は DANGER と同様のデザインです）。
IMPORTANT
非常に重要な指示。

HTML エクスポート

「HTML保存」を実行すると、以下のような特徴を持つドキュメントが生成されます。

  • 目次の自動生成: ドキュメント内の見出し（h1〜h4）を抽出し、サイドバーにナビゲーションを表示します。
  • テーマ固定: 保存時のテーマ（ライト/ダーク）がそのまま適用された状態で出力されます。
  • 完全スタンドアロン: 生成されたファイル一つで全てのスタイルとスクリプトが完結します。
