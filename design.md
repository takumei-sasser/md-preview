# MD · Preview デザイン再設計 (stitch)

このドキュメントは、現在の Markdown プレビューツールを **stitch** (モダン、プレミアム、高密度なデザイン) で再構築するための設計ガイドラインです。

## 1. デザインコンセプト： "The Architect's Workspace"
- **雰囲気**: ダークモードを基調とした、エンジニア・ライターが没入できる「コックピット」のような洗練された作業空間。
- **キーワード**: 精密 (Precision)、流動性 (Fluidity)、ガラスモフィズム (Glassmorphism)、微細な反応 (Micro-interaction)。

## 2. カラーパレット (Modern Dark / Vibrant Accents)
- **Primary BG**: `#05070a` (漆黒よりも深い紺)
- **Surface**: `#0f111a` (カードやパネルの背景)
- **Border**: `rgba(255, 255, 255, 0.08)` (極薄の境界線)
- **Accent 1 (Focus)**: `#3e63dd` (インテリジェンスな青)
- **Accent 2 (Success)**: `#00e5a4` (活力のあるエメラルド)
- **Text (Main)**: `#eef1f5`
- **Text (Mute)**: `#8892b0`

## 3. 主要なUIコンポーネントの刷新案

### 3.1 統合ヘッダー (Glass Header)
- `backdrop-filter: blur(12px)` を使用した半透明のフローティングヘッダー。
- ボタンはアイコン＋テキストの構成から、ホバー時にのみラベルが出る「ミニマルモード」も検討。
- ステータスインジケーター（保存状態、接続状態など）を右端に配置。

### 3.2 エディタエリア (Minimalist Editor)
- 行番号エリアとエディタ本体を、フォントサイズや行間の微調整で「プロ用ツール」の質感に。
- フォーカスされている行のハイライト。
- 入力時の微かなサウンド（オプション）や、タイピングに伴うスムーズなスクロール。

### 3.3 プレビューエリア (Paper Canvas)
- プレビュー部分は「浮いた紙」のように見せる。
- 背景がダークでも、ドキュメント部分は読みやすさを重視し、コントラストを最適化したダークテーマ。
- **Mermaid図面**: 背景に薄いグリッドを敷き、設計図らしさを強調。

### 3.4 下書き管理 (Card Grid / Tray)
- 単なるリストではなく、サムネイル（または冒頭のプレビュー）付きのカード形式。
- 右側からスライドインするドロワー形式に変更し、作業を中断せずに切り替え。

## 4. アニメーション・マイクロインタラクション
- **パネルリサイズ**: 分割線のドラッグ時に、プレビューが「ヌルヌル」と追従する（requestAnimationFrameを使用）。
- **保存通知**: 画面中央下部から「ふわっ」と浮き上がり、数秒で粒子のように消えるトースト。
- **ボタン**: クリック時に微かな `scale(0.95)`。

## 5. stitch 用の追加機能案
- **集中モード (Zen Mode)**: ヘッダーやツールバーを隠し、エディタとプレビューのみにする。
- **フォント切り替え**: インターネット接続時には Google Fonts (Inter, JetBrains Mono 等) を動的にロード。
- **エクスポート設定**: HTML保存時に、テーマのカスタムカラーをその場で選べるダイアログ。

## 6. 技術的な実装方針
- **CSS変数**: すべての色、サイズ、影を `:root` で管理。
- **構造**: `display: flex` と `grid` を組み合わせた柔軟なレイアウト。
- **互換性**: 標準機能（Vanilla JS）を維持しつつ、演出部分に最新の CSS 機能（Subgrid, Container Queries 等）を検討。
